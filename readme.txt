== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click “upload” link.
3. Choose the file “Vendeur.zip”, locate the theme at “theme/vendeur.zip” from the downloaded package.
4. Click “Install Now” button
5. Now you can activate the theme from “Appearance -> Themes” Menu

== Setting Homepage ==
1. Create a new page
2. Choose Homepage as for template, save
3. Go to widget manager, and populate arrange desired widget in "Homepage Widget Area" 
4. This widget area is full width
5. Recommended widget to put "Vendeur-Hero Images", "Vendeur-Product", "Vendeur-Products Categories" and "Vendeur - Recent Post"
6. Go to Setting - Reading
7. Pick the page as homepage
